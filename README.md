# Injeplast New - Website Institucional

> **Status: Projeto Privado**
> Este repositório está mantido para fins de consulta técnica e histórico de desenvolvimento.
> É uma LongPage funcional, usada pela empresa atual do desenvolvedor.

## Sobre o Projeto
Desenvolvimento da plataforma digital para a **Injeplast New**, empresa focada no tratamento e industrialização de moldes plásticos. O projeto prioriza a clareza técnica e o rigor estético, refletindo a precisão da engenharia aplicada na fábrica.

## Especificações Técnicas
* **Layout:** Implementação de grid estruturado em 3 colunas para a seção de serviços e notícias.
* **Responsividade:** Adaptação completa para diferentes dispositivos via CSS Flexbox e Grid.
* **Dinâmica:** Manipulação de DOM e lógica de interface via JavaScript Vanilla.

## Estrutura de Diretórios
```text
├── assets/
│   └── img/           # Logotipo e ativos de marca
├── css/               # Estilização e sistemas de grid
├── js/                # Lógica e comportamento dinâmico
└── index.html         # Estrutura principal
```

## Contato e Propriedade
Projeto de propriedade da **Injeplast New**. O conteúdo reflete os serviços de industrialização, polimento e manutenção técnica de moldes.
