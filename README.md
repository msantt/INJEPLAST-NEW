## 🏢 Sobre o Projeto
Plataforma digital desenvolvida para a **Injeplast New**, empresa focada no tratamento e industrialização de moldes plásticos. O projeto reflete a precisão da engenharia aplicada na fábrica, priorizando clareza técnica e rigor estético em cada detalhe da interface.

## 🚀 Tecnologias e Integrações
As tecnologias foram escolhidas para garantir performance e uma experiência de usuário fluida:

* ![HTML5](https://img.shields.io/badge/html5-%23E34F26.svg?style=for-the-badge&logo=html5&logoColor=white)
* ![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=for-the-badge&logo=css3&logoColor=white)
* ![JavaScript](https://img.shields.io/badge/javascript-%23F7DF1E.svg?style=for-the-badge&logo=javascript&logoColor=black)
* **EmailJS API:** Integração para captura de leads e envio de formulários via client-side.

## 🛠 Especificações Técnicas
* **Layout:** Grid estruturado em 3 colunas para as seções de serviços e notícias.
* **Responsividade:** Adaptação completa via CSS Flexbox e Grid Layout.
* **Interatividade:** Manipulação de DOM e lógica de interface via JavaScript Vanilla.
* **Formulário Dinâmico:** Sistema de envio com feedback de estado ("Enviando...") e validação de campos.
